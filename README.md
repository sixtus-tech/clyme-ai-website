# Clyme AI Website

AI-Powered Websites, SEO & Lead Generation

## Build. Rank. Convert. On Autopilot.

This is the official website for Clyme AI, showcasing our AI-powered services for website development, SEO optimization, and lead generation.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
